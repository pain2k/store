# Программа Магазин
## Автор
*Рамиль Талипов*

## Описание программы
Магазин с фильмами и книгами

## Параметры запуска
Программа написана на языке [Ruby](https://ru.wikipedia.org/wiki/Ruby).
Для запуска программы необходимо зайти в директорию программы и выполнить команду:
```
ruby main.rb
```
